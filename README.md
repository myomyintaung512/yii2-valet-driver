# yii2-valet-driver
Yii2 Valet Driver

> install 

```
cd ~/.config/valet/Drivers/
wget https://raw.githubusercontent.com/myomyintaung512/yii2-valet-driver/master/Yii2ValetDriver.php
```

## yii2-app-basic

```
valet link app-name
http://app-name.test
```

## yii2-app-advanced

```
valet link app-name
http://app-name.test
http://app-name.test/backend
```
